# Data-Compression-App
An app designed to compress data

For Linux/Mac please go to Actions/Rulare din start pentru Windows/Linux/Mac, then scroll to the bottom of the page and download the artifact according to your OS and decompress the downloaded folder. After that, enter the terminal in the respective folder and run the following commands:

#LINUX
chmod +x executabile/CompresorHuffman_linux
chmod +x executabile/DecompresorHuffman_linux

#MAC
chmod +x executabile/CompresorHuffman_mac
chmod +x executabile/DecompresorHuffman_mac

then run the app:
java -jar Huffman_Compiler.jar
